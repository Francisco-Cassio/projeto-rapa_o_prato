# 🍽️ Rapa o Prato

> **Página web conceitual para conectar comunidades e combater o desperdício de alimentos.**

O **Rapa o Prato** é uma página web institucional desenvolvida como um projeto de testes por estudantes de Análise e Desenvolvimento de Sistemas do IFPI. O objetivo do projeto é apresentar uma interface baseada em uma ideia de negócios. A nossa ideia funciona como uma ponte inteligente entre estabelecimentos comerciais e o consumidor final, permitindo que alimentos excedentes do dia (em perfeito estado de conservação) sejam anunciados com preços reduzidos.

## 🛠️ Tecnologias Utilizadas

O projeto foi construído utilizando as tecnologias fundamentais do ecossistema Web:

- **HTML5:** Estruturação semântica do conteúdo da página.
- **CSS3:** Estilização customizada através do uso de fontes corporativas e posicionamento de elementos.
- **JavaScript:** Implementação da lógica de comportamento da interface e rolagem da página.
